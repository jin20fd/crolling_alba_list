# 데이터 크롤링 기반 구인구직 정보 통합 데이터베이스 구축
알바몬, 알바천국 두 플랫폼의 사이트를 크롤링하여, 아르바이트 구인구직 정보 데이터 수집

두 플랫폼이 가지고 있는 공통적인 속성을 분류하여 구인구직 정보의 검색 편의성을 증진시킨 구인구직 정보 데이터베이스를 설계 및 구축
